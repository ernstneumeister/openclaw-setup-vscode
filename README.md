# openclaw-setup-vscode

Starter-Kit für die Einrichtung von OpenClaw über VS Code mit einem externen Coding-Agent.

## Enthalten
- `AGENTS.md` – zentrale Arbeitsanweisung für den Agenten
- `CLAUDE.md` – Wrapper für Claude Code
- `START-PROMPT.md` – kurzer Einstiegs-Prompt für Skool / Lektionen
- `SETUP-PROMPT.md` – ausführlicher Setup-Prompt für die eigentliche Einrichtung
- `.env.example` – Vorlage für lokale Zugangsdaten
- `LICENSE` – Lizenzhinweis
